# Blue Coding App

## Description

This app has the idea of make a URL shortener API at the same time that fetch information about it.

## Requirements

You'll need to have the following packages:

- [NestJS CLI](https://www.npmjs.com/package/@nestjs/cli)
- [NodeJS](https://nodejs.org/) on v21.4.0 (you can use NVM if you need)

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
